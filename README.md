# Hearth & Hound

A premium dog-wellness storefront (calming + orthopedic comfort). Single-file static
site — no build step, no dependencies.

## Run locally
Open `index.html` in any browser. That's it.

Or serve it:
```powershell
npx serve .
```

## What's inside
- Responsive storefront: hero, trust bar, 8-product catalog
- Quick-view product modals
- Working cart drawer (quantities, free-shipping threshold, subtotal)
- Reviews, About, and newsletter sections

## Deploy
Drop the folder on any static host — Vercel, Netlify, Cloudflare Pages, or GitHub Pages.

## Go-live checklist
1. **Payments** — replace the demo checkout with Shopify / Stripe / Lemon Squeezy.
2. **Supplier** — connect a dropship supplier (Zendrop, Spocket, CJdropshipping) and
   swap the placeholder SVGs for real product photos.
3. **Domain** — point a custom domain at the host.
